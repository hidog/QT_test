========================================================================
    MAKEFILE 專案：VS2008 專案概觀
========================================================================

AppWizard 已經為您建立了這個 VS2008 專案。  

這個檔案包含一份摘要，簡要說明構成 VS2008 專案的
所有檔案，它們個別的內容。


VS2008.vcproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔。 
    它含有產生該檔案之 Visual C++ 的版本資訊，以及有關使用應用程式精靈
    所選取之平台、組態和專案功能的資訊。

這個專案可以讓您藉由呼叫在精靈中輸入的命令，從 Visual Studio 進行
建置/清除/重建。建置命令可以是 nmake 或是您使用的任何其他工具。

這個專案並不包含任何檔案，所以在 [方案總管] 中不會顯示任何的檔案。
